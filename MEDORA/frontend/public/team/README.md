# Team photos

Place team headshots here. They appear on the About page.

**Expected filenames:**
- `shareef.jpg` — Nagur Shareef shaik
- `srikakani.jpg` — Sri Kakani
- `mandava.jpg` — Aasrith Mandava
- `asish.jpg` — Asish Gogineni

Images are displayed as 96×96px circles. Use `.jpg`, `.jpeg`, or `.png`.
